# Data System Timeline

A visual chronology of major data-system breakthroughs from the 1960s through 2026, covering key people, papers, software systems, companies, and the concrete problems each breakthrough solved.

## Features

- 83 milestones across 5 historical eras and 25 technical categories
- Full-text search and category/era filtering
- Responsive design with collapsible filters on mobile
- Single-file HTML — no build step, no dependencies

## View

Live at **https://data-system-school.github.io/data-system-timeline**
